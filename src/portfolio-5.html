Burguer Queen

Información del proyecto

  • Tecnologías usadas: HTML5, CSS3, React, y consumo de API HTTP.
  • Demo del proyecto: Aquí

El objetivo de la aplicación es ser un sistema para el personal de Burguer Queen, donde los meseros pueden agregar y eliminar fácilmente pedidos a una orden y el jefe de cocina puede ver los pedidos que llegan en orden.

Esta aplicación web está diseñada como un mock para la empresa Chilis, por lo que seguimos la línea gráfica en el diseño. Además entrevistamos a un extrabajador de la empresa para evaluar las necesidades reales a la hora de tomar pedidos.

Desarrollamos el proyecto con React, creando y reutilizando componentes efectivamente. Para trabajar con el servidor, hicimos peticiones HTTP mediante fetch hacia un mock de una base de datos con el programa Mockoon.
