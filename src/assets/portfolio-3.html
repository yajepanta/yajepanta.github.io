Innova Social

Información del proyecto

  • Tecnologías usadas: HTML5, CSS3, Javascript y Firebase
  • Demo del proyecto: Aquí

Esta aplicación web está diseñada como un mock para el colegio Innova Schools. El objetivo es facilitar la comunicación entre colegas sobre temas estrictamente profesionales, evitando así que los profesores usen redes sociales como Facebook o Whatsapp, por lo que el diseño es similar para facilitar la familiaridad del usuario.

Permite el registro de usuarios con un correo y contraseña, e iniciar sesión con Gmail. Dentro de la red social, los usuarios pueden crear, eliminar posts y subir imágenes.

Usamos Cloud Firebase como base de datos para almacenar los posts y datos de usuario, y Cloud Storage para almacenar las imágenes.
